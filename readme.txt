POKEMON WORLD GAMES
"Games survival role-playing bertemakan Pokemon"

Setiap pemain akan masuk kedunia Pokemon,
dan akan dibantu oleh Doraemon the Pokemon expert.
Terdapat pokemon yang hilang didunia pokemon, 
dan para pemain perlu menangkap Pokemon liar yang berkeliaran.
Setiap kali mendapatkan pokemon pemain akan semakin kuat 
dan dapat mengendalikan pokemon tersebut.
Permainan semakin seru dengan hadirnya legendary pokemon!
Jika pemain mengalahkan seluruh legendary pokemon. Pemain akan menang.

Apa yang dibutuhkan?
Seluruh file game Pokemon ini, dan GNU Prolog untuk menjalankan games.

Bagaimana menjalankan games ini?
	1. Consult file main.pl di GNU Prolog
	2. Selanjutnya jalankan command start. untuk memulai permainan.
	3. Setelah Homepage dari game muncul, kamu dapat menjalankan games
	   sesuai keinginanmu.
	4. Jika kamu bingung, command apa saja yang dapat dijalankan,
	   jalankan command help. Help akan memberikan informasi tentang
	   command yang dapat kamu jalankan.
	5. Kamu dapat menikmati permainan, sesuai arahan yang
	   yang diberikan di games Pokemon world ini!
